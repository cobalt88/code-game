How it works

Code quiz is comprised of 10 multiple choice questions.
Each question is worth 10 points.
You will have 120 complete the quiz.
Each incorrect answer will subtract 10 seconds.
Game is over when the timer hits zero.
High scores will be displayed at the end of each round.

Hit the start button whenever you are ready to begin.
