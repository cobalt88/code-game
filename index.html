Welcome to Code Quiz!

How it works

Code quiz is comprised of 10 multiple choice questions.
Each question is worth 1 point.
You will have 60 seconds to answer each question.
If the timer reaches zero before you select an answer it will be counted as incorrect.
High scores will be displayed at the end of each round.

Hit the start button whenever you are ready to begin.
